# 📚 Enrollment Application

A full-stack enrollment application with RSVP functionality, developed during my training period as part of academic evaluation. The project focuses on managing user registrations and event participation efficiently, integrating a responsive frontend with a Spring Boot backend and Swagger for API documentation and testing.
